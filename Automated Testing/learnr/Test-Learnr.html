Motivation

gradethis was throwing errors as of 10/04 when using many of their functions. This is a temporary solution until the bugs are addressed.

Basically, the function checker() can be used as in the tutorial_options function under the argument exercise.checker. The function checks items using an internally stored list defined under comment 5. Users would need to adapt this list if they want to test with different questions. Only results are graded and not the code itself. I also added additional calls that saves quiz takers results in a RDS file (see comment 2). This can be removed or commented out if not wanted.
